# E-commerce-Order-management

Problem Statement:

Develop a simple E-Commerce Management System .
The system should manage:

- Product details
- Cart operations
- Order placement
- Payment processing

It should allow users to:

- View product details
- Apply discount
- Update stock
- Add/remove items from cart
- Place orders
- Process payments



Approach / Logic Used:

The project is implemented using Python classes and OOP concepts.

Classes Used

1. Product

Handles:

- product details
- stock update
- discount calculation
- displaying product information

2. Cart

Handles:

- adding products to cart
- removing products
- calculating total amount
- clearing cart

3. Order

Handles:

- placing order
- cancelling order

4. Payment

Handles:

- payment processing
- payment status tracking

Logic Flow

1. User enters product details
2. Menu options are displayed
3. User selects operation:
   - Product details
   - Add to cart
   - Place order
   - Payment
4. Corresponding class methods are executed



Technologies Used

- Python
- Object-Oriented Programming (OOP)
- Git
- GitHub
